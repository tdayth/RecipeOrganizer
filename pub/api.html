API Documentation

Getting started

To get started, you want to add the following lines in your HTML file: 

        <link rel="stylesheet" type="text/css" href="css/website.css">
        <link rel="stylesheet" type="text/css" href="css/RecipeOrganizer.css">
        <script defer type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js">
        <script defer type="text/javascript" src="js/RecipeOrganizer.js" charset="utf-8">
        <script defer type="text/javascript" src="js/examples.js" charset="utf-8">

Then to initialize an object, use: 

        const recipe = new RecipeOrganizer();

Functions

.createDessertTemplate(element, divCol, desCol, oneCol, TwoCol)

Creates a dessert template. Adds this template to element. The arguments divCol, desCol, oneCol, TwoCol correspond to the background colours of the main box, description box, and ingredient box, respectively.


.createStandardTemplate(element, divCol, desCol, oneCol)

Creates a standard template. Adds this template to element. The arguments divCol, desCol, oneCol correspond to the background colours of the main box, description box, and ingredient box colours, respectively.


.makeTitle(title)

Sets the title of the recipe.


.addImage(image)

Adds image to recipe.


.addDescription(description)

Adds description of recipe to template.


.addIngredient(type, ingredient, volume, measurement)

Takes in the type of ingredient (normal, dry, wet) depending on which template is used (dessert, standard). Then it takes volume, measurement which will be stored in the event that the user makes conversions. all of these will be added to the ingredient list on the template.


.addScale()

Adds the option for the user to scale ingredients.


.addVolume()

Adds the option for the user to change the volume measurement.


.addInstructions(tranCol, instCol)

Creates a div box in which steps of a recipe can be added. The arguments tranCol, instCol are the transition, and instructions background colours, respectively.


.addSteps(step, stepCol)

Adds step to the instruction list. The argument stepCol is the background colour of each step.


.expandIngredients(px)

Expanding the ingredient box by px pixels.


.expandDescriptionBox(px)

Expand the description box by px pixels.


.removeShadow()

Removes all shadows on the template.


.removeOutline()

Removes all outlines on the template.
