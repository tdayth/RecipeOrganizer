Examples

These are examples on how you could use this library. First, let's consider a dessert template: 

            const recipe = new RecipeOrganizer();

            // Give an HTML element to to add the library to.
            const thisElement = document.querySelector("#dessertExample");
            recipe.createDessertTemplate(thisElement, "pink", "white", "white", "white");

Then we can add information to the template as such: 

            recipe.makeTitle("New York Style Cheesecake");
            recipe.addImage("./cheesecake.jpg");
            recipe.addDescription("Cheesecake that you can easily make at home!");
            recipe.expandIngredients(120);
            recipe.addIngredient("wet", "Cream cheese packages", "1000", "g");
            recipe.addIngredient("wet", "Condensed milk", "500", "ml");
            recipe.addIngredient("dry", "Flour", "0.75", "cup");
            recipe.addIngredient("wet", "Sour cream", "0.75", "cup");
            recipe.addIngredient("wet", "Ricotta", "1", "cup");
            recipe.addIngredient("wet", "Vanilla extract", "2", "tsp");
            recipe.addIngredient("wet", "eggs + yolk", "3", "");

We can also add a volume conversion, volume scaler, and step-by-step instructions to the template as such: 

            recipe.addScale();
            recipe.addVolume();
            recipe.addInstructions("violet", "skyBlue");
            recipe.addSteps("Mix creamcheese until softened in the mixer, \
            then add ricotta. Then as its mixing, add condensed milk and vanilla extract.", "white");
            recipe.addSteps("Then add 1 egg at a time while it is slowly mixing.", "white");
            recipe.addSteps("Sift the flour while mixing. Then after, add the sour cream.", "white");
            recipe.addSteps("Finally, bake at 400° for 10 minutes. Then 225° at 25 minutes.", "white");
